## 🌍 Wanderlust

Wanderlust represents the deep desire to explore new places, experience different cultures, and discover the world beyond everyday life.
This project is inspired by that spirit of exploration — helping users find, share, and experience unique stays and travel destinations.

---

## 🏠 Airnb – Full Stack Travel Stay Platform

This project is a **full-stack Airbnb-style web application** that allows users to browse, create, and manage property listings inspired by real travel experiences.

It is designed using the **MVC architecture** to ensure scalability, maintainability, and clean code structure.

---

## ⚙️ Tech Stack

### 🧩 Backend

* **Node.js** – Server-side runtime environment
* **Express.js** – Web application framework for routing and middleware handling
* **MongoDB** – NoSQL database for storing listings and user data
* **Mongoose** – ODM for MongoDB schema modeling and data validation

### 🎨 Frontend

* **EJS (Embedded JavaScript Templates)** – Server-side rendering
* **EJS-Mate** – Layout and partial template support
* **CSS** – Styling and responsive UI

### 🔧 Additional Tools & Middleware

* **Method-Override** – Enables PUT & DELETE requests in forms
* **Nodemon** – Auto-restart server during development
